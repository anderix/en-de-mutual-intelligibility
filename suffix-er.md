# Suffix -er (agent)

German agent *-er* maps directly to English agent *-er*. A noun formed this way denotes a person or thing that does the action of the verb. The pattern is identical to English; only the underlying verb differs.

For person nouns, the feminine form takes *-erin* (*Lehrer / Lehrerin*). The source wordlists generally show only the masculine; the feminine is implicitly part of the same vocabulary.

## Pairs

**driver ↔ der Fahrer** *(A1, from fahren)*
*Der Fahrer wartet schon vor dem Hotel.*

**teacher ↔ der Lehrer** *(A1, from lehren)*
Note: English *teacher* and German *Lehrer* are not etymologically related, but the suffix pattern is the same.
*Unser Lehrer erklärt das sehr gut.*

**printer ↔ der Drucker** *(A1, from drucken)*
Both noun-from-verb. Also a tech device in both languages.
*Der Drucker funktioniert heute nicht.*

**receiver ↔ der Empfänger** *(A1, from empfangen)*
*Der Empfänger wohnt in Berlin.*

**seller ↔ der Verkäufer** *(A1, from verkaufen)*
*Der Verkäufer hilft mir bei der Wahl.*

**renter / landlord ↔ der Vermieter** *(A1, from vermieten)*
*Vermieter* is the one who rents OUT (landlord). The renter (tenant) is *Mieter*.
*Mein Vermieter ist sehr freundlich.*

**student / pupil ↔ der Schüler** *(A1, from Schule)*
*Die Schüler kommen aus verschiedenen Ländern.*

**one who errs / mistake ↔ der Fehler** *(A1, from fehlen)*
This one's a trap: *Fehler* means "mistake/error" — it's an *-er* noun but not an agent in the usual sense.
*In diesem Text gibt es einen Fehler.*

**worker / laborer ↔ der Arbeiter** *(A1, from arbeiten)*
*Die Arbeiter beginnen um sieben Uhr.*

**customer ↔ der Kunde** *(A1)*
*Kunde* is irregular — not formed with *-er* — but follows the agent semantics. Listed here as a near-miss worth noting.
*Jeder Kunde bekommt einen kleinen Kaffee.*

## Pairs to expect at higher levels

Almost any German verb can spawn an agent noun: *Maler* (painter), *Bäcker* (baker, already in A1 as *Bäckerei*), *Sänger* (singer), *Spieler* (player), *Käufer* (buyer), *Leser* (reader), *Hörer* (listener / phone receiver), *Schreiber* (writer/scribe). The pattern is fully productive in both languages.

## Note

The Bäckerei pattern: German *-erei* maps to English *-ery* for "place where the *-er* works." *Bäcker / Bäckerei* = *baker / bakery*. Same logic gives *Brauerei / brewery*, *Schreinerei / carpentry workshop*, *Druckerei / print shop*.
