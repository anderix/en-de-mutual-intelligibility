# p → pf / ff

The High German consonant shift turned Proto-Germanic */p/* into:
- **pf** word-initial or after a consonant
- **ff** between vowels or word-final

English kept the older */p/*.

## Pairs

**apple ↔ der Apfel** *(A1)*
After vowel, so *pf*.
*Der rote Apfel schmeckt süß.*

**open ↔ öffnen** *(A1, verb)*
Between vowels, so *ff*.
*Bitte öffne das Fenster.*

**sleep ↔ schlafen** *(A1, verb)*
Between vowels, so *ff* in some forms (*Schlaf*, *schliff* historically); in *schlafen* the *p* has fully shifted but the spelling shows *f*.
*Sie schläft bis neun Uhr morgens.*

## Pairs to expect at higher levels

*pepper → Pfeffer* (initial *pf*), *pipe → Pfeife*, *pound → Pfund*, *plant → pflanzen* (initial cluster *pfl*), *ship → Schiff* (final *ff*), *help → helfen* (already in A1 as *helfen*, the medial *p* → *f*), *deep → tief*, *cheap → kaufen* (irregular), *hope → hoffen*.

## Note

Word-initial *p* in modern German often signals a Latin/Greek loanword (*Park, Plan, Polizei, Problem, Preis*), since the shift only applied to native Germanic vocabulary. So a German word starting with *p* (not *pf*) is usually a loanword, while a German word starting with *pf* is usually native Germanic.
